# Smart Attendance Management System

## Overview

Smart Attendance Management System is an AI-powered attendance tracking platform designed for educational institutions. The system uses Face Recognition Technology, Automated Faculty Assignment, Classroom Monitoring, and Real-Time Analytics to manage attendance efficiently.

## Key Features

- Face Recognition Attendance
- Automatic Faculty Assignment
- Live Student Count Monitoring
- Empty Classroom Detection
- Student Attendance Tracking
- Faculty Attendance Management
- HOD Dashboard
- Principal Dashboard
- HR Management
- Chairman Dashboard
- Leave Management
- Attendance Analytics
- Report Generation
- Role-Based Access Control
- Email & SMS Notifications
